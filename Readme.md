# Garment Manufacturing Analytics Dashboard (Power BI)

## 📌 Project Overview
This project analyzes production, quality, downtime, and cost performance across multiple garment factories using Power BI.

## 🏭 Business Problem
Manufacturing managers need visibility into factory efficiency, defect rates, downtime causes, and production costs to improve operational performance.

## 🛠 Tools & Technologies
- Power BI
- Power Query
- DAX
- Microsoft Excel

## 📊 Dashboard Features
- Executive overview with KPIs
- Factory-wise performance comparison
- Product quality & defect heatmap
- Downtime & maintenance analysis
- Cost & profitability insights
- Interactive slicers and drill-through

## 📁 Repository Structure
- `Garment_Manufacturing_Dashboard.pbix` – Power BI report file
- `data/` – Sample dataset used
- `screenshots/` – Dashboard preview images

## 🚀 How to Use
1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Refresh data if required
